# Installation
## Install dependencies:
cd ui  
npm install

## Techstack
React  
Typescript  
EmotionJS  
UI  
Jest  
Axios

## Setup environment variables:
Create a .env file in the root directory and add your environment variables:  
.env  
This file contains environment-specific settings. Here's a sample:  
REACT_APP_API=
